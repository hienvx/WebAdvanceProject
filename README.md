# Cài đặt code trên máy local
1. Cài đặt Nodejs (bắt buộc)
2. Mở powershell chạy lệnh
`npx lerna bootstrap`
3. Triển khai ứng dụng 
`npx lerna run start`
(hiển thị log ở terminal:
`npx lerna run --parallel start`)
